ENUNCIADO (DESCRIPCIÓN)






Recordemos que una hoja de estilos (CSS) es un conjunto de reglas que los navegadores utilizan para interpretar como deben mostrar los diferentes elementos de un documento HTML.

CSS es una herramienta que permite al diseñador realizar una unión de estilo (fuentes, tamaño, color, espacios, etc.) y añadirlo a un documento HTML.

Las diferentes formas de incluir estilos CSS en un documento HTML:

1) Usando una etiqueta LINK que llama a una CSS externa.

2) Usando la etiqueta <STYLE> dentro de la <HEAD>

3) Utilizando una CSS importada denominada @import.

4) Un atributo de style usado dentro de <BODY>

Recordemos también que para enlazar una hoja de estilo externa en un documento HTML se realiza a través de la etiqueta <link>, que debe posicionarse dentro de la sección <HEAD> con los atributos rel, type, href, y title definidos para conseguir que los estilos se carguen de forma adecuada.

Un documento HTML podrá contener tantas hojas de estilo como sean necesarias.

Los archivos de hojas de estilo son simples documentos de texto guardados con la extensión .css.

Para importar una hoja de estilo se utiliza la directiva @import y solamente puede utilizarse dentro de una etiqueta <style> dentro de la sección <HEAD>.
