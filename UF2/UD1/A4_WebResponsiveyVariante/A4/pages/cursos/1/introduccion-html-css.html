Introducción

Te has interesado por andar el camino para hacer webs y has llegado hasta aquí con un dilema: ¿por dónde empiezo? Estás en el sitio adecuado.

En esta página vas a encontrar la manera de empezar paso a paso, como hice yo, desde el principio. Despacito y con buena letra...

HTML

¿Qué es el HTML? Pues, haciendo un símil, HTML es dibujar o maquetar, organizar los espacios de la web, marcando bien la estructura.

Esto se consigue dominando el "idioma" del HTML, bueno el lenguaje y, como cualquier idioma, tiene sus normas y sus convenciones.

¿Qué es el CSS?

Si el HTML era hacer el dibujo o maquetar la web, el CSS nos va a permitir colorearla.
